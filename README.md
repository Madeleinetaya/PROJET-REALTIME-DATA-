# PROJET-REALTIME-DATA-
L’objectif  est de concevoir et d’implémenter un pipeline de traitement de données en streaming pour un cas d’usage e-commerce. 
Le système doit : 
-Ingérer des événements utilisateurs via Kafka
-Traiter ces événements en Event Time avec Apache Flink
-Détecter des anomalies de comportement
-Etre entièrement reproductible.
