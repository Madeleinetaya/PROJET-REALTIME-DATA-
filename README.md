# PROJET-REALTIME-DATA-
Projet de realtime data. L’objectif  est de concevoir et d’implémenter un pipeline de traitement de données en streaming pour un cas d’usage e-commerce. Le système doit : ingérer des événements utilisateurs via Kafka, traiter ces événements en Event Time avec Apache Flink, détecter des anomalies de comportement, être entièrement reproductible.
